Concerto

Bienvenue sur Concerto, notre plateforme d’apprentissage musical pour robot, où l’on « emprunte » le talent de la foule pour éduquer nos créations. Le fruit du travail de nos volontaires est diffusé ici même. Si vous voulez interagir avec le groupe en direct, suffit d’appuyer sur le bouton «Lancez l'expérience!» en dessous. Le stream en bas de la page est là en dépannage, si le stream pricipal est plein (ou planté!). Si vous vous sentez perdu, consultez les boutons guides. Bonne expérience!
